# Backend (Laravel 11)

> Este directorio se llena al ejecutar `./scripts/bootstrap-backend.sh` en Fase 1.

Hasta entonces este directorio existe pero está vacío intencionalmente. El script crea aquí:

- Proyecto Laravel 11 con dependencias del producto.
- Estructura de carpetas DDD (`app/Domain/<Context>/`).
- Configuración de Sanctum, Reverb, Horizon, Spatie Permission y Activity Log.
- Tests con Pest.

Ver detalles en [`docs/POS_MAESTRO_v3.md`](../docs/POS_MAESTRO_v3.md), Parte XII (DevOps), y [`CONTRIBUTING.md`](../CONTRIBUTING.md), sección "Estilo de código".
