# Architecture Decision Records (ADR)

Los ADRs del proyecto viven en un unico directorio en la raiz del repositorio:

    /docs/adr/

Ahi estan el indice, la plantilla (`_template.md`) y todos los ADRs numerados
(0001 en adelante), incluido el 0007 (TenantContext + Octane) que originalmente
se redacto aqui y se consolido en la raiz.

Este directorio se mantiene solo como puntero para no romper referencias
historicas. No agregar ADRs nuevos aqui: usar `/docs/adr/`.
